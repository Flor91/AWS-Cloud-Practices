Práctica de DynamoDB

En esta práctica vamos a ver como utilizar este servicio de AWS para crear tablas y hacer consultas.

Aquí sugerimos utilizar nuevamente un notebook (Jupyter por ejemplo) para ir ejecutando el código secuencialmente. También se puede utilizar cualquier otro entorno que cuente con Python.

Deberán tener instalado boto3. Tutorial

Tarea 1: Crear una tabla

Ingresar al servicio de DynamoDB y crear una tabla.

Configurar el nombre de la tabla como practice-[SUFFIX] (donde suffix es el primer nombre del alumno y la primer letra del apellido) y user_id como primary key.

Tarea 2: Insertar datos desde Python

Para esta parte de la práctica se sugiere usar un notebook de Jupyter para ir ingresando los comandos secuencialmente.

import boto3

# Obtener la referencia a la tabla:

dynamodb = boto3.resource('dynamodb', region_name='us-east-1')

table = dynamodb.Table('practice-[SUFFIX]')

# Mostrar la fecha de creacion de la tabla

print(table.creation_date_time)

# Insertar item.

table.put_item(

   Item={

        'user_id': '123456',

        'username': 'janedoe',

        'first_name': 'Jane',

        'last_name': 'Doe',

        'age': 25,

        'account_type': 'standard_user',

    }

)

table.put_item(

   Item={

        'user_id': '1222',

        'username': 'juanperes',

        'first_name': 'Juan',

        'last_name': 'Peres',

        'age': 55,

        'account_type': 'admin',

    }

)

Tarea 3: Consultar datos por clave principal

Hacer una query para obtener los datos del usuario con id 123456:

response = table.get_item(

    Key={

        'user_id': '123456',

    }

)

item = response['Item']

print(item)

“””

Resultado esperado:
{'username': 'janedoe', 'last_name': 'Doe', 'account_type': 'standard_user', 'user_id': '123456', 'first_name': 'Jane', 'age': Decimal('25')}

“””

Tarea 4: Consultar datos usando filtros

Para hacer esto disponemos de dos facilidades para hacer queries:

  • Key: Usando el índice primario o los secundarios.
  • Atributo: Por cualquier campo de los items.

Hacer una query para obtener los datos del usuario con id 1222:

from boto3.dynamodb.conditions import Key, Attr

response = table.query(

    KeyConditionExpression=Key('user_id').eq('1222')

)

items = response['Items']

print(items)

“””
Resultado esperado:

[{'username': 'juanperes', 'last_name': 'Peres', 'account_type': 'admin', 'user_id': '1222', 'first_name': 'Juan', 'age': Decimal('55')}]
“””

Hacer una query para obtener todos los usuarios de menos de 55 años:

response = table.scan(

    FilterExpression=Attr('age').lt(55)

)

items = response['Items']

print(items)

“””
Resultado esperado:

[{'username': 'janedoe', 'last_name': 'Doe', 'account_type': 'standard_user', 'user_id': '123456', 'first_name': 'Jane', 'age': Decimal('25')}]
“””

Combinar los dos filtros anteriores y ver que resultado da:

response = table.query(

    KeyConditionExpression=Key('user_id').eq('1222'),

    FilterExpression=Attr('age').lt(50),

)

items = response['Items']

print(items)

“””
Resultado esperado:

[]
“””

En este último caso se aplicó un filtro sobre el resultado de la query por key que eliminó los resultados de usuarios menores de 50.

Tarea 6: Utilizar en nuevo índice en una query

response = table.query(

    IndexName='age-index',

    KeyConditionExpression=Key('age').eq(55),

)

items = response['Items']

print(items)
