Tarea 2: Importar datos desde RDS con sqoop

Para hacer esto nos vamos loguear al nodo master (instancia de EC2)  a través de SSH y vamos a correr los siguiente comandos:

$ sudo -u hadoop hive -e "create database pablol"

$ sudo -u hadoop sqoop import-all-tables \

  --connect jdbc:mysql://database-2.cjkg6mo85h0r.us-east-2.rds.amazonaws.com/classicmodels \

  --username=admin \

  --password=gor0AO6FfgmFsNAVI6N2 \

  --hive-database pablol \

  --compression-codec=snappy \

  --as-parquetfile \

  --warehouse-dir=/user/hive/warehouse \

  --exclude-tables offices,productlines,products \

  --hive-overwrite \

  --hive-import
